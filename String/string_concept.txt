1.string is a sequence of character used to represent text

    -> create string

       - let str = "karan";

    -> string length

       - str.length

    -> str indices

       str[0], str[1], ....

    -> Template literals 

       - a way to have embedded expression in string 

       - (string interpolation): to create strings by doing substitution of placeholder

       -  `string text ${expression} string text` syntax

    -> string methods 

       - there are built in function to manipulate string

       - str.toUpperCase()
       - str.toLowerCase()
       - str.trim(): remove whitespace
       - str.slice(start,end?): return part of string  --> abcd  --> slice(1,3)  --> print "bc"
       - str1.concat(str2): joins str2 with str1
       - str.replace(searchVal, newVal)
       - str.charAt(idx)
       
       - strings are immutable means not change old string , make copy of old string and and add the new value in them , then update old string.
